В доме «Пушкинская 15» пять лет не работает ни один механизм контроля собственников за ТСЖ

По закону органы управления ТСЖ переизбираются раз в два года, каждая смета утверждается общим собранием, а работа правления ежегодно проверяется ревизией. Ни один из этих трёх механизмов сейчас не работает — это не мелкая формальность, а системный риск для дома.

Без обвинений — только сопоставление нормы закона и факта, с документами по каждому пункту.

Высший орган управления ТСЖ — общее собрание членов; для решений ему нужен кворум больше половины голосов дома. Без участия большинства собственников ТСЖ как форма управления нежизнеспособно — не по чьей-то отдельной вине, а по конструкции самого закона.
